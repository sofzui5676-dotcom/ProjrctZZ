# Трекер привычек

PWA-приложение для отслеживания ежедневных привычек с поддержкой темной темы и визуализацией прогресса.

## Функциональность
- Управление профилем пользователя
- Добавление/удаление привычек
- Отметка выполнения привычек
- Просмотр статистики за 7 дней и за выбранную дату
- Визуализация прогресса в виде графиков
- Светлая и темная тема
- Установка как PWA на телефон

## Технологии
- HTML5, CSS3, JavaScript (ES6)
- Docker + Nginx
- PWA (Service Worker, Manifest)
